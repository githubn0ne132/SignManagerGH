# SignManagerGH
*Spécification du projet de génération de signatures Outlook (version Python hébergée) : L'objectif est de développer une solution automatisée pour la génération et la configuration des signatures Outlook, avec une interface d'administration dédiée au service marketing. Stack technique proposée : - Backend : Python avec un framework type Django ou Flask, pour la gestion de l'interface web, l'authentification et la génération des signatures avec la librairie Pillow. - Frontend : HTML, CSS et JavaScript pour une expérience utilisateur intuitive. - Base de données : SQLite pour une solution simple et efficace pour le stockage des modèles et des données. Points clés : Le service marketing pourra directement modifier et mettre à jour les éléments visuels de la signature, garantissant la saisonnalité et la cohérence de la marque, simplifiant ainsi le processus de déploiement.
